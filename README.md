# Batman Keychain

This project features a Batman-inspired keychain designed using Onshape as part of a mechanical design assignment.

## Project Details
- Designed in Onshape
- Created based on a reference image with estimated dimensions
- Extruded to a thickness of 2 mm
- Includes a 4 mm keyring hole
- Exported as an STL file for 3D printing
